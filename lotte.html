Lotte Library

Lotte Library is a single page website that allows users to view, search, reserve books that they are keen in reading. Users can view the books available in the library, see the description of the books and filter their search by genre, and author. A reserve and bookmark button allows users to star and reserve the book they are interested in. Users are also able to view the library stats to see which are the most popular genre or book format, that the library readers are keen in reading the most. In addition, users are able to view the library's inventory as well to find out more about the book. There is a contact form for the users to contact the library in any case of enquiries that they might have.

This website was built using html, fullpage.js, css, and anime.js etc.
