Movies, Dreamland Theatres

This is a movie website coded using PHP and mySQL as the database. This website allows users to register, login, edit their information, as well as add, delete and edit their own reviews for each movie. The movies in the list are also added through the use of PHP and mySQL and through the creation of this website, I have learnt how back-end of a website would function using local-side database, in this case, mySQL and using styling to make it user-friendly for my users to get around the site.
